# 🧘 MindEase – Emotion-Rich Mental Health Chatbot

MindEase is a colorful, interactive mental wellness chatbot that responds with empathy,
emotions, and warmth using NLP and rule-based intelligence 💙.

## ✨ Highlights
- Multiple emotion detection
- Supportive & gentle responses
- Emoji-rich conversations
- Calming gradient UI
- Ethical non-clinical design

## 🛠️ Built With
- Python
- Flask
- NLTK
- HTML & CSS

## 🚀 Run Locally
pip install -r requirements.txt  
python app.py

Open: http://127.0.0.1:5000/

## ⚠️ Disclaimer
MindEase is not a medical professional.
It provides emotional support only.

## 🌱 Purpose
Technology with empathy matters.
